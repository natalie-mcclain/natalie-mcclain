# Hi, I'm Natalie 👋

I'm a second-year PhD student in Neuroscience at the **Icahn School of Medicine at Mount Sinai**, working in the [NARC Lab](https://narc.lab) with Dr. Rita Goldstein. My work in the lab focuses on how large-scale networks reconfigure dynamically in response to naturalistic stimuli (e.g., movies), and how the underlying white-matter architecture may constrain those brain dynamics.


## 🧠 Research Interests

- Naturalistic neuroimaging paradigms
- Real-time fMRI neurofeedback 
- Whole-brain modeling and brain state dynamics
- Network control theory
